## Run Locally

Retrofit Library

```bash

    implementation 'com.squareup.retrofit2:retrofit:2.9.0'

```

Gson Library

```bash
  implementation 'com.squareup.retrofit2:converter-gson:2.9.0'
```

Image Loading Library 

```bash
  implementation 'com.squareup.picasso:picasso:2.8'
```
